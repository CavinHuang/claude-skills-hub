## 📋 项目简介

Claude Skills Hub 是一个开源的AI技能共享平台，为Claude用户提供丰富的预构建技能和工具。无论您是开发者、设计师、商务人士还是学生，都能在这里找到适合您的AI技能。

### 🎯 核心价值

- **🔧 开发效率**: 专业的代码生成、测试和开发工具
- **🎨 设计创新**: UI组件生成、设计系统创建工具  
- **💼 商业智能**: 商业文档、数据分析报告生成
- **📊 数据洞察**: 数据可视化、分析报告工具
- **🎓 学习辅助**: 教育内容、知识问答系统

## 🚀 快速开始

### 环境要求

- **Claude Desktop**: v1.0 或更高版本
- **Node.js**: v18.0 或更高版本（开发环境）
- **Git**: 版本控制工具

### 安装技能

1. **浏览技能市场**
   ```
   访问 https://claudeskill.site
   浏览和搜索您需要的技能
   ```

2. **选择技能**
   ```
   点击技能卡片查看详情
   阅读技能说明和使用方法
   确认技能适合您的需求
   ```

3. **一键安装**
   ```
   点击"安装技能"按钮
   等待安装完成
   在Claude中开始使用
   ```

### 开发技能

1. **Fork 项目**
   ```bash
   git clone https://github.com/YOUR_USERNAME/claude-skills-hub.git
   cd claude-skills-hub
   ```

2. **创建技能**
   ```bash
   npm run create-skill <skill-name>
   cd skills/<skill-name>
   ```

3. **测试技能**
   ```bash
   npm run test
   npm run build
   ```

4. **提交贡献**
   ```bash
   git add .
   git commit -m "feat: add new skill"
   git push origin feature/new-skill
   ```

## 🏗️ 技术架构

### 前端技术栈
- **Next.js 14**: React全栈框架
- **TypeScript**: 类型安全的JavaScript
- **Tailwind CSS**: 实用优先的CSS框架
- **Framer Motion**: 流畅的动画库

### 后端技术栈  
- **Node.js**: JavaScript运行时
- **PostgreSQL**: 关系型数据库
- **Redis**: 缓存和会话存储
- **Prisma**: 现代数据库ORM

### 部署架构
- **Vercel**: 前端部署平台
- **AWS**: 云服务基础设施
- **GitHub Actions**: CI/CD自动化
- **Docker**: 容器化部署
